
# 🍽️ Flavour Quest

A modern food discovery web application built using the MERN stack.

---

## 🚀 Features
- 🔐 User Authentication (Login/Register)
- 🍲 Add & Explore Recipes
- 🔍 Search Recipes
- 📸 Image Upload
- 📱 Responsive Design

---

## 🛠️ Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB
