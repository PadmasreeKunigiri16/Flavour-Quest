# 🍽️ Flavour Quest

A modern food discovery web app built using MERN stack.

## 🚀 Features
- 🔐 User Authentication
- 🍲 Add & Explore Recipes
- 🔍 Search Recipes
- 📸 Image Upload

## 🛠️ Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB

## 📂 Project Structure
recipe-finder/

## ⚙️ Installation
git clone https://github.com/PadmasreeKunigiri16/Flavour-Quest.git
cd Flavour-Quest
npm install
npm run dev
