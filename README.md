# 🍽️ Flavour Quest

A modern food discovery web app built using MERN stack.

## 🚀 Features
- 🔐 User Authentication (Login/Register)
- 🍲 Add & Explore Recipes
- 🔍 Search Recipes
- 📸 Image Upload

## 🛠️ Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB

## 📂 Project Structure
client/
server/

## ⚙️ Installation
git clone https://github.com/PadmasreeKunigiri16/Flavour-Quest.git
cd Flavour-Quest
npm install
npm run dev
